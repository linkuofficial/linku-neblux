極點與步階響應

同樣推一下，有的系統穩穩停住，有的卻要來回震盪好幾下才安靜。

決定它是哪一種的，只是一張圖上一個點的位置。把那個點在圖上挪一挪，右邊「被推一下之後的反應」就跟著換了個性——過衝、震盪、慢慢拖，全寫在這個點放在哪裡。

這一頁只想讓你看見一件事：一個系統會不會過衝、震盪，全由它的極點放在複數平面（左邊那張座標圖）上的哪裡決定。

互動主體

挪動極點，看它怎麼安定下來

拉下面的滑桿，左圖那個 ✕ 就跟著動；右圖「被推一下的反應」立刻換個性。你會看見：點放在哪，就決定它穩不穩、晃不晃。

極點：系統的性格座標　·　✕ 就是極點，它放哪，脾氣就怎樣

步階響應：把它推一下之後，怎麼反應

觀察任務

先讓兩張圖回答

  1. 把 ζ 從大慢慢調到小：白線從「乖乖爬到 1」變成「衝過頭，再回頭晃幾下」。第一個高峰從 ζ 多少開始冒出來？
  2. 停在會晃的地方，只動 ωn：曲線形狀一模一樣，變的只有時間軸的秒數，左圖那個點往外跑。同一種個性，換了個節拍。
  3. 把 ζ 加大越過 1：晃動不見了，但到位反而變慢、拖尾。找出「剛好不再晃」的那個點。

形式化

剛剛看到的形狀，寫成方程式

H(s) = ωn² / ( s² + 2·ζ·ωn·s + ωn² )

poles (0<ζ<1)：s = −ζ·ωn ± j·ωn·√(1 − ζ²) = −σ ± j·ω_d

step (underdamped)：y(t) = 1 − ( e^(−ζ·ωn·t) / √(1 − ζ²) ) · sin( ω_d·t + φ )， ω_d = ωn·√(1−ζ²)， φ = cos⁻¹ ζ

overshoot (0<ζ<1)：M_p = exp( −ζ·π / √(1 − ζ²) )

把游標移到算式裡的符號上（手機點一下），看它對應到畫面上的哪個東西。

極點放在哪裡，決定系統是哪一種個性：

  • ζ = 0 → 無阻尼（極點在虛軸上，等幅震盪永不停）
  • 0 < ζ < 1 → 欠阻尼（過衝後收斂，ζ 越小衝越高、晃越久）
  • ζ = 1 → 臨界阻尼（重極點落在負實軸上，不震盪、最快到位）
  • ζ > 1 → 過阻尼（兩個實極點，不震盪，但被較慢的那個拖尾）

每個符號對回剛剛畫面上的東西：

ζ
ζ 阻尼比：cos θ = ζ（θ 從負實軸量起），決定過衝與震盪。
ωn
ωn 自然頻率：極點到原點的距離＝圓半徑，決定快慢；只縮放時間軸，不改變曲線形狀。
s
s：複數頻率，s 平面上的座標；極點＝讓分母為零的位置＝畫面上的 ✕。
σ = ζωn
σ = ζωn：極點實部的大小＝衰減率，點越靠左，衰減越快。
ω_d = ωn√(1−ζ²)
ω_d = ωn√(1−ζ²)：極點虛部＝系統實際震盪的頻率。
φ = cos⁻¹ ζ
φ = cos⁻¹ ζ：相位，讓 y(0) = 0 對得起來。
e^(−ζωn t)/√(1−ζ²)
e^(−ζωn t)/√(1−ζ²)：teal 虛線包絡線，震盪被關在這條線裡面，逐漸收攏。
y(t)
y(t)：白線，追著穩態值 1 跑的實際反應。
